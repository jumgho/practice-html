!DOCTYPE ________
: 문서의 타입을 나타내는 문법으로 최상단에 사용.

html, /html
: html문서의 head와 body를 감싸는 고위층 태그. 사람간의 규칙

head, /head
: body를 설명하는 상단 부분을 규정하는 태그. 사람간의 규칙

body, /body
: 본문의 내용을 규정하는 태그. 사람간의 규칙

title, /title
: 웹페이지의 이름을 표시하는 태그, head 부분에 들어감. title을 통해 해당 웹사이트가 '이미지'가 아닌 '정보'로 간주하게 하는 중요한 태그.

meta
: 브라우져가 해당 웹사이트를 제공할 때 어떤 방식의 파일로 열어서 제공할 지 인지시키는 태그.
ex- mata charset="utf-8"

h1,h2,~,h6 , /h1,h2,~,h6
: HTML의 heading(제목)을 표시하는 태그. 숫자로 제목의 크기를 설정한다.

ol, /ol, ul, /ul
: 각각 ordered list, unordered list라는 의미로 list 태그를 감싸는 모태그. 숫자를 매겨주는지 여부에 따라 나뉜다.

li, /li
: 한개의 list를 나타내는 태그. ol 혹은 ul에 감싸져서 list를 만든다.

br
: line break, 줄바꿈을 표시하는 태그, 한 줄 바꿈

p
: pharagraph, 단락을 표시하는 태그, 단순 줄바꿈이 아닌 단락이라는 정보를 주기위해 쓰이는 태그.

a, /a
: anchor, 링크를 걸 때 쓰이는 태그. 해당 위치에 닻을 내리듯 href(hypertext+reference)와 함께 사용하여 링크를 거는 태그.
target="_blank" : 링크가 새탭으로 뜨게 하는 정보 태그.
title="_______" : 링크의 툴팁을 나타내게 하는 정보 태그.
ex- a href="https://www.youtube.com/watch?v=V3pkC1hE-as&list=PLuHgQVnccGMDZP7FJ_ZsUrdCGH68ppvPb&index=14" target="_blank" title="생활코딩 유튜브"

strong, /strong
: 글씨를 진하게 표시하는 태그.

u, /u
: underline, 밑 줄 태그.

img
: 이미지 삽입 태그.
